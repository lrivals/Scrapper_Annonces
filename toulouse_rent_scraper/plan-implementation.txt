## 🧩 PHASE 4 — Scraper SeLoger (Playwright)

### 4.1 Rôle du scraper

* Charger la page avec filtres
* Attendre le rendu JS
* Extraire :

  * lien annonce
  * prix
  * localisation
  * surface (si dispo)

---

### 4.2 Logique Playwright (simplifiée)

```python
from playwright.sync_api import sync_playwright

def scrape_seloger(url):
    results = []

    with sync_playwright() as p:
        browser = p.chromium.launch(headless=True)
        page = browser.new_page()

        page.goto(url, timeout=60000)
        page.wait_for_selector("article")

        cards = page.query_selector_all("article")

        for card in cards:
            try:
                link = card.query_selector("a").get_attribute("href")
                price = card.inner_text()
                results.append({
                    "site": "SeLoger",
                    "link": link,
                    "raw": price
                })
            except:
                continue

        browser.close()

    return results
```

👉 **Lien = priorité absolue**, les autres infos peuvent être enrichies ensuite.
